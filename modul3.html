HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat dan menyusun halaman web. HTML menggunakan berbagai tag dan elemen untuk menentukan struktur dan konten dari sebuah halaman web, seperti heading, paragraf, gambar, link, dan banyak lagi. HTML bekerja sebagai kerangka dasar sebuah website yang kemudian dapat dipercantik dengan CSS dan dibuat lebih interaktif dengan JavaScript.

CSS (Cascading Style Sheets) adalah bahasa style sheet yang digunakan untuk mengatur tampilan dokumen HTML. CSS memungkinkan kita untuk mengontrol layout, warna, font, spacing, dan berbagai aspek visual lainnya dari halaman web. Dengan CSS, kita dapat memisahkan konten (HTML) dari presentasi, membuat website lebih mudah dikelola dan lebih menarik secara visual.
